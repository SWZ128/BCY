# BCY原图
用于获取[半次元](https://bcy.net/)无水印原图的Tampermonkey脚本

## 安装
- 从 Greasy Fork 中安装
https://greasyfork.org/zh-CN/scripts/434023

## 使用
启用该脚本后，在图片下方会出现一个「查看原图」的超链接，点击可打开查看相应图片的无水印原图。
